# Password-Generator

This is a simple and responsive Password Generator web application built using HTML, CSS, and JavaScript.  
It allows users to generate strong and secure random passwords instantly.

## Features

- Generate random secure passwords
- Fast and lightweight
- Clean and simple UI
- Works in browser only
- No backend required

## Tech Stack

- HTML
- CSS
- JavaScript

## How to Use

1. Open index.html in browser
2. Click Generate button
3. Password will be created instantly

## Purpose

Strong passwords are important for security.  
This project generates random passwords using JavaScript to help users create safe credentials.
